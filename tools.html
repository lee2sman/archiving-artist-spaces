Tools

This is an in-progress list of tools for archiving artist-run spaces’ online activities such as: scraping websites, ‘freezing’ and preserving facebook and similar closed proprietary platform event pages, and more.

Archive-It - Internet Archive’s institutional tool to backup the website/data of organizations

Wayback Machine Downloader - Download an entire website from the Internet Archive Wayback Machine

Conifer - Rhizome’s web archiving service

Web Recorder: Web Archiving For All. Webrecorder provides a suite of open source tools and packages to capture interactive websites and replay them at a later time as accurately as possible

Wget - CLI free software package for retrieving files using HTTP, HTTPS, FTP and FTPS

instagram scraper - CLI python program

flickr-scrape by sam levine and others
